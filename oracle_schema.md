# Oracle Database Table Schema: `IFS_TICKETS`

This schema is designed to hold the ticket data currently stored in `NA Daily work.xlsx`.

## Table: `IFS_TICKETS`

| Column Name | Data Type | Constraint | Description |
| :--- | :--- | :--- | :--- |
| `TICKET_ID` | `NUMBER` | `PRIMARY KEY` | Internal ID (Sequential) |
| `TICKET_NO` | `VARCHAR2(50)` | `UNIQUE` | ServiceNow Ticket Number (e.g., SCTASK...) |
| `REQ_NO` | `VARCHAR2(50)` | | Request Number |
| `CONTACT` | `VARCHAR2(100)` | | Requester/Contact Name |
| `TEAM_MEMBER` | `VARCHAR2(100)` | | Assigned Team Member |
| `DATE_CREATED` | `DATE` | `NOT NULL` | Creation Date |
| `STATUS` | `VARCHAR2(20)` | | Current Status (Open, Closed, etc.) |
| `CATEGORY` | `VARCHAR2(100)` | | Ticket Category |
| `DESCRIPTION` | `CLOB` | | Full Request Details/Description |
| `TYPE` | `VARCHAR2(50)` | | Ticket Type |
| `PIC` | `VARCHAR2(100)` | | Person In Charge (Assign To) |
| `DATE_RECEIVED` | `DATE` | | Time - Arrive |
| `DATE_RESOLVED` | `DATE` | | Time - Close |
| `REMARKS` | `CLOB` | | Additional Comments/Remarks |
| `CREATED_BY` | `VARCHAR2(50)` | | System Audit User |
| `LOAD_DATE` | `DATE` | `DEFAULT SYSDATE` | When the row was imported |

## SQL Script (Create Table)

```sql
CREATE TABLE IFS_TICKETS (
    TICKET_ID      NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    TICKET_NO      VARCHAR2(50) UNIQUE,
    REQ_NO        VARCHAR2(50),
    CONTACT       VARCHAR2(100),
    TEAM_MEMBER    VARCHAR2(100),
    DATE_CREATED   DATE NOT NULL,
    STATUS        VARCHAR2(20),
    CATEGORY      VARCHAR2(100),
    DESCRIPTION   CLOB,
    TYPE          VARCHAR2(50),
    PIC           VARCHAR2(100),
    DATE_RECEIVED DATE,
    DATE_RESOLVED DATE,
    REMARKS       CLOB,
    CREATED_BY    VARCHAR2(50),
    LOAD_DATE     DATE DEFAULT SYSDATE
);

CREATE INDEX IDX_IFS_TICKETS_STATUS ON IFS_TICKETS(STATUS);
CREATE INDEX IDX_IFS_TICKETS_DATE ON IFS_TICKETS(DATE_CREATED);
```
